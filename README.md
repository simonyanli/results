# How to add results

## AIIO

1. Go to `_data/aiio/` and add a new `resultsXX.csv` (or tsv, or yml etc.).
1. Go to `aiio/` and add a new `aiioXX.md`.
1. Open `_data/aiio/aiio.yml` and add a new entry.
